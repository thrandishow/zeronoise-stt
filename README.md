# Русский Speech-to-Text API

Микросервис для распознавания русской речи на основе модели Whisper. Предоставляет REST API для преобразования аудио в текст с поддержкой различных форматов аудиофайлов.

[![FastAPI](https://img.shields.io/badge/FastAPI-005571?logo=fastapi)](https://fastapi.tiangolo.com/)
[![Whisper](https://img.shields.io/badge/Whisper-green?logo=Ghostty)](https://github.com/openai/whisper)

## Особенности

- **Поддержка русского языка** - распознавание русской речи
- **Автоматическая конвертация** - поддержка MP3, WAV, M4A, OGG, FLAC и других форматов
- **Валидация файлов** - автоматическая проверка формата и размера аудио
- **Полная документация** - интерактивная документация Swagger UI

## Требования

- Python 3.11 ( Whisper работает с этой версией Python, исходя из требований из репозитория Whisper! )
- macOS/Linux (Windows не тестировался)
- Для MacBook Air с чипами M1/M2/M3: macOS 12.0+
- 4+ ГБ свободной памяти для загрузки модели
- 500 МБ свободного места на диске для модели и зависимостей

## Установка

### 1. Клонируйте репозиторий

```bash
git clone https://github.com/thrandishow/zeronoise-stt
cd zeronoise-stt
```

### 2. Создайте виртуальное окружение

```bash
python3 -m venv .venv
source .venv/bin/activate  # Для Linux/Mac
.venv\Scripts\activate  # Для Windows
```

### 3. Установите зависимости

#### Рекомендуемый способ ( требуется менеджер пакетов **uv**)

```bash
pip install uv
uv sync
```

## Запуск сервиса

```bash
uvicorn main:app --host 0.0.0.0 --port 8000
```

## Ограничения

1. **Размер файла:** Максимальный размер аудиофайла - 25 МБ
2. **Длина аудио:** Для файлов длиннее 30 секунд рекомендуется предварительное разделение на сегменты
3. **Производительность:**
   - MacBook Air M1/M2: ~2-3 секунды на 1 секунду аудио
   - Сервер с NVIDIA GPU: ~0.5-1 секунда на 1 секунду аудио
4. **Качество распознавания** зависит от:
   - Четкости дикции
   - Фоновых шумов
   - Акцента говорящего
